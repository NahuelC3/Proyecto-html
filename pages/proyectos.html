Mis Proyectos

En esta sección vas a encontrar los trabajos prácticos y proyectos personales que vengo desarrollando a lo largo de mi formación como programador.

Proyecto 1: Portafolio Personal Web

Este mismo sitio web representa mi primer gran hito técnico. Fue desarrollado aplicando una arquitectura limpia de carpetas y siguiendo los estándares más modernos de la industria.

  • Tecnologías: HTML5 Semántico, CSS3 avanzado, Bootstrap 5.
  • Enfoque: Diseño Mobile-First responsivo para celulares, tablets y computadoras.

Proyecto 2: Repositorio de Prácticas Git

Un espacio dedicado al control de versiones donde gestiono ramas, commits y subidas de código, clave para trabajar de forma profesional o en equipo.

  • Tecnologías: Git, GitHub Desktop, Markdown para documentación.
  • Enfoque: Buenas prácticas en el flujo de trabajo de repositorios públicos.
